prebacivanje u drugi branch (git checkout ime_brancha)
- git checkout develop

u kojem sam branchu?
- git branch ili git status

stvaranje neke datoteke (relativno beskorisno)
- git add basic.txt

kad napraviš sve potrebno i dodaš u folder eventko di treba bit
- git add .  -> kopira cijeli folder sa svim promjenama u njemu
- git commit -m "Dodan originalni Eventko logo"
- git push

kad se prebaciš u novi branch i želiš updateati svoj folder na kompu tako 
da sadrži sve nedavno dodane stvari
- git pull
